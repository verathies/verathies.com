What it's all about

Rogue Pickings is a catering company which delivers lunch right to your desk. Every meal is made with fresh, organic ingredients and recipes are carefully picked. Check out their daily specials and enjoy a delicious break!

Work method

I built this website from a Photoshop PSD. Starting with a sketch of the HTML structure, document flow, links and text were added. Then I created the CSS file to make the website looked just like the PSD.

Future ideas

This website can be improved further with responsive design, this way it will adapt to different screens. Additional pages can be created to extend the menu options or a menu builder could be added.
